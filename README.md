# 2026 Expo NativeWind Template

本テンプレートは、**Expo SDK 54**、**NativeWind v4**、**Biome**、および **Bun** を活用した、2026年の開発スタンダードを見据えた高品質な React Native 開発環境です。

パフォーマンス、デザイン性、および保守性を兼ね備えたアプリケーションを効率的に開発するために、最新のツールセットを最適化して構成しました。

---

## 主要な機能

- **Expo SDK 54**: 最新の安定版 Expo。強力なエコシステムを最大限に活用可能です。
- **React Compiler & New Architecture**: `app.json` にて有効化済み。次世代のパフォーマンスを提供します。
- **NativeWind v4**: Tailwind CSS の直感的なスタイリングを React Native で実現し、開発効率を向上させます。
- **Expo Router (src/app)**: Next.js ライクなファイルベースルーティングにより、直感的な画面遷移を構築可能です。
- **Biome**: 高速なリンターおよびフォーマッターにより、コードの品質を高い水準で維持します。
- **Bun**: パッケージ管理からビルドまで、開発プロセスのすべてを高速化するランタイムです。

---

## プロジェクト構造

```text
.
├── assets/             # アイコン、スプラッシュ画像等の静的リソース
├── src/
│   └── app/            # アプリケーションのコアロジック（Expo Router）
│       ├── (tabs)/     # タブ画面の定義
│       ├── _layout.tsx # 共通レイアウトの定義
│       └── global.css  # グローバルスタイルの定義
├── biome.json          # Biome の設定ファイル
├── tailwind.config.js  # Tailwind CSS の拡張設定
└── package.json        # プロジェクトの依存関係
```

## クイックスタート

### 1. 依存関係のインストール
**Bun** を使用して、必要なパッケージをインストールします。

```bash
bun run install
```

### 2. 開発サーバーの起動
シミュレーターまたは実機を用いてアプリケーションを起動します。

```bash
# 開発サーバーの起動
bun run start

# プラットフォームを指定して起動
bun run android  # Android
bun run ios      # iOS
```

---

## コード品質の維持

Biome を活用することで、一貫性のあるクリーンなコードベースを維持できます。

```bash
# コードの検証（Lint）
bun run lint

# コードの自動修正およびフォーマット
bun run lint:fix
bun run format
```
